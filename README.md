## 功能介绍

当群里开始+1时，机器人也会参与其中（默认为每+1了3次，要么跟着一起+1要么打断施法）。

包括普通消息，QQ表情，还有图片（表情包）。

图片（表情包）使用了base64编解码。

是个群聊都可能会复读。

## 用法简介

按常规方法导入插件即可。

## 已知问题

特殊qq表情无法复读，例如没有未来的撕梓咩等那种画师的原创表情包。
